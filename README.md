Student Productivity Toolkit
A C-based productivity application designed for students to efficiently manage tasks, track study time, compute GPA, record notes, and use a Pomodoro timer.
This project consolidates essential academic utilities into a single lightweight terminal program written entirely in C.
Features
1. Task Manager
-Add tasks
-View all tasks
-Mark tasks as complete
-Store deadlines
-Saves and loads tasks automatically

2. Study Session Tracker
-Log study sessions with subject name
-Automatically timestamps entries
-Saves sessions to a study log file

3. GPA Calculator
-Add course grades
-Enter credit hours
-Computes cumulative GPA
-Saves course data to disk

4. Quick Notes
-Create short notes
-Automatically timestamps every note
-View saved notes
-Saved permanently in a text file

5. Pomodoro Timer
-Standard 25-minute session
-Custom duration option
-Simulation for console-friendly usage

6. Data Persistence
-Stores tasks and course data using binary files
-Notes and study logs saved in text format
-Automatically loads data when the program starts
